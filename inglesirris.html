About the Page

1What is ConsideraFly?

ConsideraFly is a website that enables users to perform a set of essential actions. Firstly, they can obtain information about individual airports, including data such as codes, names, cities, countries, latitudes, and longitudes. Similarly, users have the ability to identify the 10 airports whose shortest paths from a given airport are the longest, along with the associated distances. Finally, it allows users to find the shortest path between two selected airports, visually displaying this route on an interactive map and providing details of each intermediate airport along the way.

2Objective

Its main objective is to construct a weighted graph based on the geographical coordinates of airports, thus modeling the connections between them. This is achieved by extracting detailed flight information from a specific dataset.

3What is its purpose?

The ConsideraFly page aims to provide an interactive and visual tool for exploring and analyzing worldwide air transportation routes, providing users with a geographic and practical perspective on air transportation routes around the world and facilitating specific analysis based on each user's needs.
